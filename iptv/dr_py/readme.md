#### 项目修复
进入容器drpy项目根目录执行以下代码
```shell
cd utils && wget https://ghproxy.liuzhicong.com/https://raw.githubusercontent.com/hjdhnx/dr_py/main/utils/update.py -O update.py
```
执行后重启9001或者docker即可正常检测升级
